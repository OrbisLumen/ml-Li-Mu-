# 感知机

## 感知机
- 给定输入$\mathbf{x}$, 权重$\mathbf{w}$,和偏移$b$, 感知机输出:
    $$o = \sigma(\langle \mathbf{w},\mathbf{x} \rangle + b) \space \space \space 
    \sigma(x) = \begin{cases} 1 & \text{if} \space x > 0 \\ -1 & \text{otherwise} \end{cases} $$ 

- 二分类: -1 或 1

## 训练感知机

```text
initialize w = 0 and b = 0
repeat
    if y_i [<w, x_i> + b] <= 0 then
        w <- w + y_i x_i
        b <- b + y_i
    end if
until all classified correctly
```
等价于使用批量大小为1的梯度下降,并使用如下的损失函数
$$\ell (y,\mathbf{x}, \mathbf{w}) = \max(0, -y\langle \mathbf{w},\mathbf{x} \rangle) $$

## 收敛定理

- 数据在半径$r$内
- 余量$\rho$分类两类
$$y(\mathbf{x}^\top\mathbf{w} + b) \geq \rho$$
- 对于$||\mathbf{w}||^2 + b^2 \leq 1$
- 感知机保证在$\frac{r^2 + 1}{\rho ^2}$步后收敛

## XOR问题

感知机不能拟合XOR函数,只能产生线性分割面
