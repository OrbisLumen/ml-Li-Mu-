# 环境配置

## conda
1. 创建环境
    ```bash
    conda create -n <name> python=3.10 -y
    ```
2. 查看已有环境
    ```bash
    conda env list
    ```
    ```bash
    conda info --envs
    ```
3. 激活环境
    ```bash
    conda activate <name>
    ```
4. 退出环境
    ```bash
    conda deactivate
    ```
5. 删除环境
    ```bash
    conda remove -n <name> --all
    ```

## jupyter
1. 打开jupyter

2. 退出jupyter
