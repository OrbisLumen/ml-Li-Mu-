# 自动求导

## 计算图
- 将代码分解为操作子
- 将计算表示为一个无环图
- 显示构造
- 隐式构造

## 两种模式
正向累积
反向累积（反向传递）

## 代码示例
**backward()**
从当前张量出发，沿计算图反向使用链式法则，把 “这个张量对各个 requires_grad=True 叶子变量的梯度” 累加到它们各自的 .grad 里

### 标量变量

```py
import torch
x = torch.arange(4.0)
x.requires_grad_(True)  # 等价于x=torch.arange(4.0,requires_grad=True)
# x.grad 默认值是None
y = 2 * torch.dot(x, x)
y.backward()
x.grad
```
输出：
```py
tensor([ 0.,  4.,  8., 12.])
```

### 非标量变量

```py
# 对非标量调用backward需要传入一个gradient参数，该参数指定微分函数关于self的梯度。
# 本例只想求偏导数的和，所以传递一个1的梯度是合适的
x.grad.zero_()
y = x * x
# 等价于y.backward(torch.ones(len(x)))
y.sum().backward()
x.grad
```
输出：
```py
tensor([0., 2., 4., 6.])
```

## 分离计算
**detach()**
```py
x.grad.zero_()
y = x * x
u = y.detach()
z = u * x

z.sum().backward()
x.grad == u
```
输出：
```py
tensor([True, True, True, True])
```

## py控制流的梯度计算

即使构建函数的计算图需要通过Python控制流（例如，条件、循环或任意函数调用），我们仍然可以计算得到的变量的梯度
